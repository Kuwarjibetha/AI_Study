# AI Learning Workspace

This repository contains Python notebooks and datasets used for learning and experimenting with data science, machine learning, and Python programming.

## Contents

- `numpy/` - NumPy examples and CSV files for array operations and data manipulation.
- `Pandas/` - Pandas practice notebooks for data analysis, aggregation, filtering, and joins.
- `Python/` - Python programming notebooks covering loops, functions, exceptions, file handling, and other fundamentals.
- `untitled folder/` - Various project notebooks, datasets, and scripts including visualization and machine learning exercises.

## Purpose

The workspace is intended for practice, exploration, and experimentation with AI-related topics, especially Python data science tools.

## How to use

1. Open the notebook you want to explore.
2. Run cells in order to reproduce the examples.
3. Add your own experiments and notes as you learn.

## Notes

- Notebooks are the main form of exploration in this workspace.
- Data files are stored alongside the notebooks for easy access.
- Feel free to organize and rename files as your learning progresses.
